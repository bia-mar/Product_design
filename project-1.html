Mobile Banking App Redesign

Lead Product Designer • June 2025 - August 2025 • Remote

Overview

A comprehensive redesign of a mobile banking application focusing on improving user experience, accessibility, and modern design patterns. The project aimed to reduce friction in common banking tasks while maintaining security and trust.

The Challenge

The existing banking app suffered from poor usability, outdated visual design, and complex navigation that frustrated users. Customer feedback indicated that simple tasks like transferring money or checking balances required too many steps.

Key challenges included:

  • Streamlining the user flow for common banking tasks
  • Modernizing the visual language while maintaining brand identity
  • Improving accessibility for users of all ages and abilities
  • Ensuring security without compromising usability

Research & Discovery

I conducted user interviews with 15 banking app users, competitive analysis of leading banking apps, and usability testing of the current application. This research revealed that users valued speed, clarity, and confidence in their transactions.

Design Solution

The redesigned interface features a card-based dashboard that surfaces the most common actions, simplified navigation with clear visual hierarchy, and improved feedback for all user actions. The color palette was refined to enhance readability while maintaining the brand's identity.

Results

Post-launch metrics showed a 40% reduction in task completion time, 65% increase in user satisfaction scores, and a 25% decrease in customer support inquiries related to app usage.
